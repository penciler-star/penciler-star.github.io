Windows下安装Python+PCL

先安装python3.6.3

PCL不好装，发现有个库叫python-pcl，装了一下午没装上。在将要放弃的时候忽然发现还有个库叫pclpy。

pclpy安装步骤

win+R，输入cmd，打开控制台，输入

1

pip install pclpy

打开python

1

python

在python中输入

1
2

import pclpy
from pclpy import pcl

done！
